# Tetris AI

My University AI Project (Tetris AI) using Python

![tetris](/screenshot/0.jpg)

## How to run project

To run the game itself you should run the following commands:

```bash
  pip install -r requirements.txt
```

and after installing requirements:

```bash
  python main.py
```
